Universal Interaction

Every component supports the universal interaction object for Power BI data behavior.

{
  "enabled": true,
  "trigger": "auto",
  "internalMode": "highlight",
  "internalScope": "self",
  "externalMode": "auto",
  "field": "__field_key__",
  "operator": "=",
  "value": "__value__",
  "selectionMode": "replace",
  "multiSelect": true,
  "showSelector": false,
  "clearOnSecondClick": true
}

ActionRequiredDescription
clearFilters—Clears all HyperPBI filters

toggleStatetargetToggles Boolean state
toggleSidebar—Toggles sidebar collapse

closeOverlaytargetCloses an overlay
toggleOverlaytargetToggles an overlay
setSteptarget, valueSets active step
nextSteptargetAdvances step
previousSteptargetReverses step

scrollTotargetScrolls to component
